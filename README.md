Nostra is a modern, responsive e-commerce website built using **HTML, CSS, and JavaScript**.

##  Project Overview

Nostra showcases fashion products with sections like **New Arrivals**, **Most Wanted**, brand highlights, promotional offers, and a newsletter subscription.  
It is designed to provide users with an engaging and smooth shopping experience.



##  Features

- Responsive navigation bar
- Offer bar with close functionality
- Banner image slider with navigation arrows
- Brand showcase section
- Customer experience highlights
- New Arrivals section
- Most Wanted products section
- Promotional offer banner
